# SET5: HyperLogLog SSSR+++ rang

Реализация вероятностного алгоритма **HyperLogLog** для оценки числа уникальных элементов в потоке данных.

## Состав проекта
- `main.cpp` — запуск экспериментов  
- `HashFuncGen.hpp` — хеш-функция (полиномиальный + универсальный хеш)  
- `RandomStreamGen.hpp` — генератор потока строк с контролируемым размером вселенной  
- `HyperLogLog.hpp` — стандартная версия HyperLogLog  
- `HyperLogLogPacked.hpp` — упакованная версия (6 бит на регистр, экономия памяти 25%)  
- `plot.py` — скрипт для построения графиков и анализа точности  

## Результаты
В репозитории сохранены:
- CSV-файлы с сырыми данными (`run_*.csv`) и статистикой (`summary_*.csv`)
- Графики и отчёты в папке `report/` для параметров:
  - `p = 12` (стандартная и упакованная версии)
  - `p = 14` (стандартная версия)

Эмпирическая ошибка согласуется с теоретической оценкой `1.04 / √m`.
